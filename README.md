# Figma Plugin Docs

Static documentation page ("What is a template?") for the BeatPitch Figma plugin,
hosted on GitHub Pages.

**Live page:** https://beatpitch.github.io/Figma-Plugin-Docs/

## Structure
- `index.html` — the page
- `images/` — screenshots & graphics
- `videos/` — demo clips (H.264 mp4)

## Updating
Edit the files, commit, and push to `main`. GitHub Pages redeploys automatically.
