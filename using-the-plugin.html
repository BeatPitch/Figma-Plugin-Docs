Typpo Figma Plugin

Using the Plugin

You've designed your template in Figma — layouts, backgrounds, text, and overlays. Now you'll run it through the Typpo plugin to turn it into animated video. Here's the whole flow, step by step.

1
Get started

Select your frames & run the plugin

In Figma, select the frame you want to turn into a template — along with any additional background assets you want available. Then run the Typpo plugin from Plugins → Typpo. It reads everything in your selection.

2
The canvas

Group your backgrounds in bg

Anything that should be treated as a background — the images or colors that sit behind your text — must live inside a group named bg. The plugin imports everything in that group as a background layer and cycles through them.

If it's not in bg, it won't be treated as a background.

3
The layers

Keep overlays outside bg

Overlays — logos, decorations, watermarks — are elements that float on top of the background. To be treated as overlays, they must live outside the bg group in your layer structure.

Overlays can be ◆ Layout scope or ◆ Template scope, and can have entrance & exit animations — see What is a Template? for how scope works.

4
The safe zone

Set the text bounding box

The bounding box of your text layer in Figma is the safe zone — the area Typpo is allowed to fill with text. Whatever size you give that box is the space text will fit into.

Make the box large and text fills it large. Make it small and the text is constrained to fit only that area. Your bounding box is your safe zone.

Safe zone Big box, big text
Large box → text fills the whole area
Safe zone Small box keeps text tucked here
Small box → text is constrained to fit inside
The dashed box is the bounding box you draw in Figma — text never spills outside it.
5
In the plugin

Choose animations per text

Inside the plugin, select any text and you'll be able to pick its animation — how it appears on screen as the transcript plays. Each text layer gets its own animation, so you can mix and match across layouts.

That's the setup

Backgrounds grouped, overlays floating, safe zones set, animations chosen. Run the plugin and Typpo composites it all into video — backgrounds cycling, your transcript animating word-by-word inside the safe zone, overlays riding on top.
