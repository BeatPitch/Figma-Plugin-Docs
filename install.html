Getting started

Install

One command in Terminal. Takes about 30 seconds. The installer downloads the latest version, copies the plugin files, and opens everything you need.

Recommended

Terminal command

Open Terminal (search "Terminal" in Spotlight), paste the command below, and press Return.

Terminal
$ bash <(curl -sL https://raw.githubusercontent.com/BeatPitch/Typpo-Figma-Plugin-Releases/main/install.sh)
1
Open Terminal — press ⌘ Space, type Terminal, hit Return.
2
Paste the command above and press Return. It downloads and installs the plugin automatically.
3
Finder opens with manifest.json highlighted. In Figma, go to Plugins → Development → Import plugin from manifest… and pick that file. (One-time only.)
4
To update later, just run the same command again.
Alternative

Download & double-click

If you'd rather not use Terminal, download the installer ZIP from the releases page.

1
Download the latest Typpo-Plugin-Installer.zip from the releases page and unzip it.
2
Right-click the file Install Typpo Plugin.command and choose Open.
3
macOS may show a security warning — click Open in the dialog. (See troubleshooting below if it blocks you.)
4
The rest is the same — Finder opens, import manifest.json into Figma once, done.

macOS security prompts

macOS protects you from unverified software. Here's what you might see and how to get past it.

⚠ "Apple could not verify…"

Shows when you double-click the .command file
Don't double-click. Instead:
  1. Right-click (or Control-click) the file
  2. Choose Open from the menu
  3. Click Open in the dialog that appears
This tells macOS you trust the file. You only need to do this once.

✕ "...is damaged and can't be opened"

Rare — usually happens with .app files, not .command scripts
Open Terminal and run:

xattr -cr ~/Downloads/Typpo\ Plugin

This strips the quarantine flag macOS put on the downloaded file. Then try opening it again.

Or skip the file entirely and use the Terminal command method above — it bypasses Gatekeeper completely.
